For interviewers

这个项目的四个重点

查看 GitHub ↗
  1. 本地优先存储Node.js 内置 SQLite 保存结构化记忆，不依赖外部数据库服务。
  2. 可追踪工作流一次模型或规则分析被组织为档案、策展和草稿三个阶段，处理快照可以回看。
  3. 检索有引用混合匹配标题、正文、人物、地点、标签与情绪，回答会回链到具体展品。
  4. 公开 Demo 隔离示例数据使用临时数据库，禁用破坏性操作，无 Key 时自动使用 Mock。
