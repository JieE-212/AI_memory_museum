      For interviewers

      这个项目的四个重点

      查看 GitHub ↗
      1. 可验证的记忆年轮每次有效修改都进入 SHA-256 父链，旧版可以核对、预览并恢复为新的当前版本。
      2. 多模态来源可回看原文、照片区域与确认文字稿各自保留来源，策展结论可以回到真实材料复核。
      3. 克制的人机协作AI、EXIF 与本机识别只生成待确认草稿，不替用户改写记忆或制造确定事实。
      4. 可体检、可携带馆藏与备份可只读验真；归档损坏零写入，脱敏版本从结构上排除私人内容。
