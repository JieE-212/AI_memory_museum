## 四、链完整性要求

- 每个 chain node 必须有 material、command、upstream、downstream、allowed output 和 must not。
- 每个 material 必须可以从 `当前候选交付审查前包索引.md` 或 `当前候选交付人工审查执行顺序总表.md` 追溯到。
- 每个 candidate 命令必须在 `package.json` 中注册，并挂入 `npm.cmd run check`。
- README 必须给出当前候选人工审查链完整性索引入口。
- 链完整性检查通过，不代表 reviewer 输出已经产生、被接收或被汇总。
